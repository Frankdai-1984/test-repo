## 自动化流程

```
每日早8点
    ↓
GitHub Actions 自动抓取热点
    ↓
生成 Markdown 摘要 → 保存到 hot-topics/
    ↓
（可选）推送到飞书/微信
    ↓
人工决策：选题 → 写作 → 审校 → 发布
```
